# AI 역학 스튜디오 탁(卓)

동양 역학 — 사주명리·자미두수·구성학·서양 점성학·주역·하락이수·당사주·숙요경·풍수·궁합·타로·전생 — 을 한자리에 통합한 단일 파일 웹앱.

## 바로 사용

웹에서 열기: **https://jeonyutak-tech.github.io/yeokhak-ai/**

또는 [명리AI.html](./명리AI.html) 파일을 내려받아 더블클릭으로 실행.

## 기능

### 사주명리
- 만세력 기반 명식 자동 생성 (절기·진태양시 보정)
- 천간·지지·오행·음양·십신 자동 분석
- 신살(도화·역마·화개·천을·양인·공망·괴강·문창·백호 등)
- 합·충·형·해·파 자동 검출
- 한난조습(寒暖燥濕) 통변 + 조후용신
- 신강·신약·중화 판별
- 대운(10년)·세운(1년)·일운(달력형) 풀이

### 보조 명리 체계
- 자미두수 명궁·신궁·복덕궁
- 구성학 본명성·월명성·일명성
- 서양 점성학 태양성좌
- 하락이수 64괘
- 당사주 12성도
- 숙요경 27숙

### 추가 모듈
- 궁합 풀이 (두 사람 통합 분석)
- 타로 카드 (3장·5장·켈틱 크로스 10장)
- 전생 해석 (자미 복덕궁·숙요 27숙·하락이수·당사주 통합)
- 지역운 (세계 국가별·한국 시·도·시·군·구·동 단위)
- 출생시간 추정 (질문·MBTI·AI 자유 텍스트)

### AI 통합
- Anthropic Claude · OpenAI GPT · Google Gemini · xAI Grok · OpenRouter · Pollinations 지원
- 학습 노트 자동 주입 — 한 번 학습한 강의 내용이 모든 해석에 반영
- 유튜브 강의 자동 학습 (Gemini, 영상 직접 분석)
- 역학 공부 탭에서 분야별 통합 정리

### 데이터 관리
- 명식·통변·해석 기록 모두 브라우저 localStorage에 자동 저장
- 내보내기 / 가져오기 (JSON)
- 검색·고정·수정·삭제

## 실행 환경

- 브라우저: Chrome / Edge / Safari / Firefox 2022년 이후 버전 권장
- 단일 HTML 파일로 작동 (외부 의존 없음, 모든 데이터·룰 내장)
- API 키는 본인 키 사용 (각 AI 공급자별)
- 오프라인 동작도 가능 (계산·표시 부분)

개인 사용 목적 자유. 상업 이용·재배포 시 문의.
